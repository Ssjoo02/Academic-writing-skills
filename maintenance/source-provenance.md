## Rules

- Prefer official venue author guidelines, official templates, CFP pages, and representative accepted papers.
- Record every source before using it in a profile.
- Use public writing advice as summarized principles, not copied text.
- Use representative papers for structure, story rhythm, evidence ordering, and reviewer-facing positioning.
- Treat official venue template, CFP, and guideline rows as source tracking for format/process constraints only unless exact reviewer guidance or representative-paper analysis is separately recorded.
- Do not copy paragraphs, distinctive phrasing, or paper-specific prose.
- If a source cannot be found or downloaded, report the missing source to the user.
