## Venue Kind

- journal

## Source Status

  - TPAMI Author Center (Computer Society): https://www.computer.org/web/tpami/author

  - IEEE Editorial Style Manual for Authors: https://journals.ieeeauthorcenter.ieee.org/wp-content/uploads/sites/7/IEEE-Editorial-Style-Manual-for-Authors.pdf

- Drift risk: medium-high; verify the current Information for Authors and manuscript-type limits before submission.

## Scope

- Applies to: TPAMI Regular papers, Survey papers, and Short papers.
- Submission version: review submission; revised/final layout limits and overlength charges are rechecked at acceptance.

## Length And Counting

- Layout: IEEE Computer Society double-column, `IEEEtran` class (`\documentclass[journal]{IEEEtran}`).
- Main text limit (manuscript types, page limits include references and author biographies):
  - Regular paper: 12 double-column pages (submissions may run up to 18 pages, subject to Mandatory Overlength Page Charges / MOPC).
  - Survey paper: 20 double-column pages.
  - Short paper: 8 double-column pages.
  - Comments paper: ~2.5 double-column pages.
- References count: yes, references count toward the page limit.
- Appendix count: appendices in the main paper count toward the limit; long proofs/extra results usually go to supplemental material.
- Abstract limit: one paragraph, 150-250 words, no citations, no numbered equations.
- Index Terms: required, alphabetical, immediately after the abstract.
- Supplementary material count: separate files only (must not be in the main PDF); no page limit on supplemental files.
- Word budget: TPAMI is page-capped (not word-capped), and references and author biographies count toward the page limit. Budget by manuscript type and weigh overlength against MOPC. See the `academic-review` skill's `references/checks/journal-submission-elements.md` (Length And Word Budget -> Page-based venues).

## Blinding

- Single-blind / non-anonymous for the standard track. Confirm the current policy, since IEEE journals occasionally pilot double-blind options.

## Revision Model

- Manuscript Central review with major/minor revision rounds; a response-to-reviewers letter is expected at revision.
- Differentiate clearly from any prior conference version in the cover letter and the paper.
- No conference-style camera-ready crunch; the accepted layout governs the final overlength arithmetic and MOPC.

## Supplementary And Extension

- Proofs, additional experiments, and extended results belong in separate supplemental files.
- For a paper extended from a conference version, state the substantial new contribution explicitly (commonly expected to be a clear, non-trivial extension).
- See `journal-vs-conference.md` for the journal drafting posture (complete story, comprehensive state-of-the-art comparison, thorough analysis).

## Post-Main Order

- Required order: Title page, Abstract, Index Terms, (optional Nomenclature), Introduction, Body, Conclusion, Appendix(es), Acknowledgment, References, (final version) Photographs and Biographies.
- Optional sections: Nomenclature, Appendices, author biographies (final version).
- Not verified: exact ethics/reproducibility statement placement for this transaction.

## Drafting Implications

- Plan a double-column page budget by manuscript type (Regular 12-14, Survey 20, Short 8) and treat references and biographies as counting toward it.
- Keep the abstract a single 150-250 word paragraph with no citations or numbered equations; add Index Terms right after it.
- Move long proofs, extra tables, and extended results to separate supplemental files, not the main PDF.
- Provide a comprehensive comparison with the state of the art; reviewers expect methodological novelty and thorough evaluation.
- Fill author and biography blocks for the final version; the standard track is not anonymous.

## Final Gate

- Before submission-ready: confirm the current TPAMI Information for Authors, manuscript type and its page limit, MOPC implications, abstract word count (150-250), Index Terms presence, supplemental-file separation, and IEEEtran compilation.
